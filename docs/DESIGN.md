# Inner Crew · 设计哲学与理论背书

> 这份文档讲三件事：① 整个产品到底在做什么、为什么值得做；② 几个关键设计细节背后的用意；③ 这些设计借了哪些心理学理论撑腰（含诚实边界）。
> 也是答辩素材库——尤其用来回应"这是玩具还是工具"。

---

## 一、一句话立意

**重要决策时，每个人心里都有一场没开成的会。Inner Crew 帮你把这场会开起来。**

我们没有发明"AI 替你做决定"。我们做的是相反的事：**把你脑内本来就在打架的几股声音，外化成看得见、听得见、会互相反驳的角色，让它们当着你的面把这场会吵完，最后由你——而不是 AI——拍板。**

AI 在这里的角色是**会议的组织者和书记**，不是决策者。这条边界是产品的伦理地基，也是它区别于"算命/聊天机器人"的关键。

---

## 二、核心隐喻：心智是一场会议

人做难决定时的真实体验，从来不是"一个理性的我在冷静权衡"，而是**好几个我在抢话**：一个喊"冲"，一个喊"别冒险"，一个惦记着别人会怎么想，一个只想知道"我到底要什么"。

把"心智是单一理性主体"换成"心智是一群声音的议事"，是 Inner Crew 全部设计的出发点。这不是文学比喻，它有扎实的理论传统（见第三节）：从 Minsky 的"心智社会"到 IFS 的"内在家庭"，把心智看作**多个部分（parts / agents / I-positions）的协作与冲突**，是一条严肃的研究脉络。

产品把这个隐喻做成了可玩的流程：
**立题 → 三步地牢（测出哪些声音在你心里更响）→ 卡片（把测量结果"指着你的行为"讲回给你）→ 会前自陈 → 决策会议三幕（声音们当面吵）→ 你裁决 → 建议书。**

---

## 三、理论地基（四根柱子）

### 1. 内在家庭系统 IFS：人格是"部分"，裁决是"自我领导"

**理论**：Richard Schwartz 的 Internal Family Systems 认为，心智天然是多重的——每个人都由许多"部分（parts）"组成（保护者、被放逐者等），同时有一个核心的"自我（Self）"，以好奇、平静、慈悲、笃定为特征。心理健康不是消灭某些部分，而是让**自我去领导（Self-leadership）**这些部分：倾听它们、不被任何单一部分挟持。

**它背书了什么**：
- 六个人格 = 六个"部分"；每个都有正当的存在意义（守护者不是懦弱，是"唯一记得疼的人"）。
- 用户的**主意识裁决 = 自我（Self）**。第三幕"听 A / 听 B / 我要综合"，就是 Self 在听完各部分后做领导。
- **AI 写死"用户裁决是最终决定、禁止重新论证"**——这正是 Self-leadership 的产品化：组织者绝不僭越去当那个 Self。
- 落败方的 `dissent`（"野心家保留意见：缓冲期是怯懦的别名"）+ 收尾"内心的会永远开不完"——对应 IFS 的核心信念：**部分不会消失，整合是持续的**，不存在"吵赢了就没有另一种声音"。

### 2. 自我抽离 & 所罗门悖论：为什么要把声音"外化成角色"

**理论**：
- **所罗门悖论（Grossmann & Kross, 2014）**：人对**别人**的难题往往比对**自己**的难题推理得更明智（聪明的所罗门王能为他人裁断，却管不好自己的人生）。
- **自我抽离（self-distancing，Kross & Ayduk）**：当人用第三人称、"墙上的苍蝇"视角看待自己的处境时，情绪调节更好、反刍更少、决策更明智。关键就是**拉开距离**。

**它背书了什么**：把"我该不该离职"这种第一人称的纠结，**重写成"六个角色在替你吵这件事"**——本质上就是制造自我抽离。你从"困在里面的当事人"变成"旁观这场会议的人"，所罗门悖论里那点"对自己的人生反而拎不清"的劣势被拉平。**外化即抽离，抽离即清醒**——这是整个外化设计最硬的科学理由。

### 3. 对话性自我 / 心智社会：多声音是合法的，不是病

**理论**：
- **对话性自我（Hubert Hermans）**：自我是一群"我-位置（I-positions）"的动态多重体，它们之间会对话、结盟、争辩——源头可追到 William James 的 I/Me、Bakhtin 的"复调"。
- **心智社会（Marvin Minsky, 1986）**：智能不是单一中枢，而是无数简单 agent 协作涌现。

**它背书了什么**：让用户安心——**脑子里有好几个声音吵架，是心智的常态，不是分裂**。产品把这件本来就在发生、却被压抑的事**做成了正常、甚至好玩的体验**。技术上也对应：六个人格各是一个 `Agent`，第二幕允许"点名呛声"，就是 I-positions 之间的对话。

### 4. 投射测验：为什么地牢这样测、为什么死活不能剧透

**理论**：投射假设（Lawrence Frank, 1939；Rorschach、Murray 的 TAT）——给人**模糊的情境**，人会把自己内在的需要与冲突**投射**上去。投射测验有效的前提是一条铁律：**被试不能知道在测什么，否则他会表演、会给社会赞许的答案，测量就失效了。**

**它背书了什么**（这是产品里一连串看似"小气"的决定的总根源）：
- **地牢用架空冒险情境**（废墟、集市、岔路），不影射现实——模糊刺激才好投射。
- **会前自陈放在地牢之后**：先郑重写下"我想要什么"再进地牢，用户会表演一致性，污染投射；先玩后问，地牢选择保持诚实。
- **地牢不显示测量轴**（去掉"现实 vs 理想"标签，换成中性地点名）：一旦告诉你这题在测理想主义，你就会照着演。
- **地牢声音匿名**（只剩颜色、名字留到会议才揭晓）、**打分隐藏、delta 与登台插话解耦**：都是为了不给用户任何"标准答案"的线索，护住测量效度。

## 四、人格为什么这样设计

### 三组根本对立轴，不是随便挑六个

```
野心家 ←→ 守护者      冒险 / 安全      （要不要赌）
共情者 ←→ 本我        他人 / 自我      （为谁而活）
计算师 ←→ 造梦师      现实 / 理想      （用什么标准衡量）
```

人生难决定，反复绞在这三对张力上：**该冒险还是求稳？该顾别人还是顾自己？该听理性的账还是听心里的画面？** 六个人格不是六种"性格类型"，而是**这三组决策张力的人格化**。对立成对，是为了第一幕"最对立的两个人格对峙"有结构来源——`pick_duelists` 取对立轴上得分之和最大的一对，让会议从你心里**最大的那道裂缝**开始。

### 为什么人格必须"极端、不中立"

每个人格的 system prompt 都被刻意写得**极端化、有固定句式、禁止中立、禁止说"看情况"**（守护者永远反问、本我永远戳破、计算师永远列条目）。理由有二：
1. **测量需要区分度**——温吞的人格盲测时分不清谁是谁，就测不出你心里哪个声音更响（我们做过盲测，要求 100% 对得上号）。
2. **决策需要把张力顶到台面**——一个会"和稀泥"的内心声音对决策毫无帮助；只有把每股冲动推到极端、让它们正面冲突，被压抑的考量才暴露得出来。第二幕硬性禁止"两边都有道理"，同理。

这也呼应一个朴素事实：你心里那个喊"冲"的声音，从不会理性地补一句"当然也要考虑风险"——它就是只管冲。**忠于这种偏执，才忠于真实的内心体验。**

## 五、关键设计细节的意义（逐条 → 理论）

| 设计细节 | 用意 | 理论来源 |
|---|---|---|
| 自陈放在地牢**之后** | 别让自我报告污染行为测量 | 投射测验·测量纯度 |
| 地牢声音**匿名**、会议才揭晓名字 | 不剧透"标准答案"；"原来一路拽我的是我的内心剧组"是一次整合的 reveal | 投射纯度 + IFS 整合 |
| 地牢**不显示测量轴**、打分隐藏 | 不让用户照着标签表演 | 投射测验铁律 |
| 计算师挂**真实检索**带数据进辩论 | 决策要接地气，对抗"内心回音室"——光靠直觉对吵会脱离现实 | 决策中的事实锚定 / 对抗确认偏误 |
| 用户裁决**最终**、AI 禁止重新论证 | 组织者不僭越当 Self | IFS·Self-leadership |
| 第三幕三个按钮都带**可选补充框** | 选边的人也有保留意见要落地，别只奖励"综合" | 尊重每个部分 |
| 发言 **👍 有道理 / 👎 不认同** | 让 Self 在会议中途就与各部分互动，而非旁观 | 对话性自我 |
| `dissent` + "内心的会永远开不完" | 承认部分不消失、整合是持续的 | IFS |
| 人格**专属色**贯穿全程 | 让六个声音成为可追踪的"角色"，支撑匿名→揭晓的连贯 | 角色一致性 / 具身记忆 |

## 六、诚实边界（很重要，也是答辩护身符）

我们**借**了 IFS、自我抽离、投射测验这些临床与实验心理学框架来**做设计的灵感和结构**，但我们：
- **不提供心理咨询/治疗**，不替代专业帮助；
- **不宣称任何临床疗效或心理测量学效度**——地牢是一个有理论启发的**体验装置**，不是经过信效度检验的量表；
- 把人格、得分、卡片都当作**叙事工具**，目的是帮你把内心的考量看清楚、把决定权交还给你，而不是给你贴一个"科学诊断"。

**坦白这条边界，本身就是产品成熟度的体现**——它把 Inner Crew 和那些假装"AI 算准你性格/替你决定"的玩意儿划清了界限。

---

## 七、一句话答辩词

> "我们没做一个替人做决定的 AI。我们把心理学里'心智是多个部分、健康在于自我领导'（IFS）和'把难题外化、拉开距离就能想得更清楚'（自我抽离 / 所罗门悖论）这两件事，做成了一个你能玩的内心剧场——**AI 只负责把那场你一直没开成的会组织起来，拍板的人始终是你。**"
