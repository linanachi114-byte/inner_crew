# Inner Crew · 演示录屏脚本（3 分钟）

> 目标：一条 3 分钟、零翻车的产品演示录屏，放简历 / README。
> 核心策略：**全程带 `?replay` 加载**——会议三幕走预录、零 LLM、每遍一致；前段漏斗的真调用只剩"试炼插话"和"自陈两问"，短而稳，翻车就重录那几秒。

---

## 0 · 录制前准备（Pre-flight）

1. 本地起服务：`uv run uvicorn main:app`（确认 `.env` 里 `STEPFUN_MODEL=step-2-16k`——非推理 ~1s，比 flash 快稳，前段漏斗更跟手）。
2. 浏览器开 **`http://127.0.0.1:8000/?replay`**。会自动：预填议题"该不该裸辞，去做自己一直想做的事" + 右下角出现 🎬 导演条。
3. 窗口调到 **16:9**（建议 1280×720 或 1920×1080），关掉书签栏、缩小到只剩页面。
4. 浏览器静音页面以外的声音；若要 BGM，项目 `music/` 里有现成 bgm，可在剪辑时叠。
5. **彩排 2 遍**：试炼插话偶发 451 审查（StepFun 误杀），命中就刷新重走那一段——这是前段唯一的不确定点，回放段绝不会出问题。

> 想更省事的"绝对零风险"版：直接点导演条 `▶ 直接开会` 跳过漏斗，只录"会议+建议书"（全程零 LLM），漏斗用截图一带而过。下面的脚本是**完整漏斗版**，更能讲清产品。

## 1 · 要打字的真实文案（照抄即可）

| 步骤 | 字段 | 直接打这段 |
|---|---|---|
| 立题 | 议题 | （已预填）`该不该裸辞，去做自己一直想做的事` |
| 自陈·计算师问"你现实里有什么牌" | 资源 | `存款够撑 8 个月，前端做了 4 年，手上有两个已经在聊的小客户。没房贷，但不敢跟爸妈说。` |
| 自陈·本我问"你到底想要什么" | 渴望 | `我怕 40 岁回头，发现自己一辈子只在替别人的梦想打工。我想做一个真正属于我自己的东西。` |
| 裁决 | 选边 | 点 **「听 野心家 的」**（不选"综合"——为了让建议书底部亮出"另一侧的声音：你没有选，但它们没有消失"这个差异点） |
| 裁决·补充（可选） | 一句话 | `但我会先留 3 个月缓冲，谈定第一个客户再走。` |

> 试炼三道门：每道门**任选一个**选项即可（选择会影响人格测量，但回放的会议内容固定，不必纠结选哪个）。建议三道门都选"先观察/再想想"那类，好让自陈问题里出现"你三道门都先观察"的个性化措辞。

## 2 · 三分钟分镜脚本（画面 × 旁白）

> 旁白可做配音，也可做字幕。语气贴项目路演稿："没开成的那场会 / 好几个你在抢话"。

| 时间 | 画面动作 | 旁白 / 字幕 |
|---|---|---|
| **0:00–0:15** 钩子 | 停在封面/立题页，议题已填好。慢慢把"该不该裸辞"这行打完（删掉重打几个字，做出"正在纠结"感） | "做一个难决定的时候，你脑子里从来不是**一个**理性的你在权衡——是好几个你，在抢话。" |
| **0:15–0:25** 立意 | 点"踏入试炼"。入场名单 6 张卡牌依次翻面亮相 | "Inner Crew 把这几个声音，变成六个会当面吵架的角色。" |
| **0:25–0:55** 试炼（测人格） | 推开第一道门 → 场景出现 → "脑内的声音正在浮现……" 插话**乱序流式**冒出来 → 选一个选项。第二、三道门**快进**（加速或跳剪），各停 2 秒 | "先走一趟试炼。你怎么选，它们就怎么吵——这一程不是测题，是在量你这个人。" |
| **0:55–1:05** 卡片结算 | "试炼结束，掉落了关于你的观察" 卡片掉落 → 点"收下，继续" | "走完，掉下几张关于你的牌。" |
| **1:05–1:25** 会前自陈 | "开会之前，两个人想先问问你" → 计算师问资源、本我问欲望 → 逐字打上面那两段回答 → 点"正式开始会议" | "开会前，计算师问你手里有什么牌，本我问你到底想要什么——它们要拿你的真话当弹药。" |
| **1:25–2:05** 第一幕·对峙 | 像素小人列队 → 野心家、守护者两位"队长"走向两侧、揭面 → 切入幕逐字陈词（这是全片**最 wow 的镜头，放慢、别跳**） | "第一幕，对峙。最针锋相对的两个你，先把话挑明。"（让画面的台词自己说话，旁白留白几秒） |
| **2:05–2:25** 第二幕·选边 | 其余四个小人"思考态" → 逐个聚焦发言、选边站队（本我站到野心家一侧）。计算师那条若带 🔍 检索标，给个特写 | "其余的声音被迫选边——没有中间地带，没有'两边都有道理'。" |
| **2:25–2:40** 裁决 | "该由你裁断了" → 点「听 野心家 的」（落败侧当场淡隐、不消失）→ 补充框打那句缓冲 → 点"生成决策建议书" | "吵到最后，谁拍板？**你**。它们只负责把会吵完。" |
| **2:40–3:00** 建议书 | 建议书展开：方向总结 / 可执行步骤 / 风险提示 / **底部"另一侧的声音——你没有选，但它们没有消失"**。缓慢下滑到底，停在这句上 | "你拿到的不是一个'算准你'的标准答案，是一份带着风险、也带着你没选的那条路的——决策建议书。内心的会，永远值得再开一次。" |

## 3 · 录制技巧

- **工具**：macOS 用 QuickTime / Mac 自带录屏（⌘⇧5）；Win 用 Xbox Game Bar 或 OBS。要做字幕/变速/跳剪用剪映或 CapCut。
- **光标**：录屏前把鼠标移动放慢；关键点击前停顿 0.5s，观众才跟得上。
- **变速**：试炼第二、三道门、长段打字，剪辑时 1.5–2× 加速，省出时间留给会议高潮。
- **导出 GIF**：建议书页有"导出会议纪要 PNG"，可在结尾点一下展示。整片可导出 MP4，再用 GIF 工具截 6–10s 高潮段（对峙镜头）放进 README 顶部。
- **README 嵌入**：把高潮 GIF 命名如 `docs/screenshots/demo.gif`，README 顶部替换占位即可（我可以帮你做这步）。

## 4 · 一句话兜底

如果只能录一个镜头：带 `?replay` 点导演条 `▶ 直接开会`，录"对峙 → 选边 → 你裁决 → 建议书"这 60 秒——零 LLM、永不翻车，就是全片的灵魂。
